# STSS D 模块 · 接口设计文档

---

## 文档说明

> ⚠️ 本文档为占位符，待 D 组补充接口设计内容

### 功能模块（参考项目要求）

1. **论坛公告** - 教师发布课程公告
2. **发帖功能** - 发表帖子、上传附件
3. **回帖留言** - 回复、交流
4. **文章管理** - 发文统计、热帖统计
5. **帖子检索** - 全文搜索

---

## 接口列表

### 帖子模块 (Posts)

| **接口名称**     | **方法** | **路径**     | **说明**                                     |
| ---------------- | -------- | ------------ | -------------------------------------------- |
| **发布帖子**     | `POST`   | `/posts`     | 支持发布讨论、提问、分享或公告，可绑定附件   |
| **获取帖子列表** | `GET`    | `/posts`     | 支持分页查询、关键字搜索、课程过滤及类型过滤 |
| **获取帖子详情** | `GET`    | `/posts/:id` | 获取详情，访问后浏览量自动 +1                |
| **编辑帖子**     | `PATCH`  | `/posts/:id` | 编辑帖子，仅作者或管理员/教师可操作          |
| **删除帖子**     | `DELETE` | `/posts/:id` | 软删除操作，仅作者或管理员/教师可操作        |

### 评论模块 (Comments)

| **接口名称**      | **方法** | **路径**              | **说明**                                     |
| ----------------- | -------- | --------------------- | -------------------------------------------- |
| **发表评论/回复** | `POST`   | `/posts/:id/comments` | 支持对帖子进行一级评论，或对已有评论进行回复 |
| **获取评论列表**  | `GET`    | `/posts/:id/comments` | 获取该帖子下的所有有效评论                   |
